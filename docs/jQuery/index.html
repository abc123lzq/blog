# 理念

jq强调的理念是写得少，做得多。压缩后大小30kb左右

js的加载完毕只能执行一次

window.onload = function () {
    alert("1");
};
window.onload = function () {
    alert("2");
};
//只会输出2

jq可以多次执行

$(doucment).ready(function () {
    alert("1")
});

//简写
$(function () {
    alert("2")
});

// 1 2 都能弹出

链式编写代码风格有助于可以读性

$(document).ready(function(){
	$(".level1 > a").click(function(){
	    //给当前元素添加"current"样式
		$(this).addClass("current")
		//下一个元素显示   
		.next().show()       
		//父元素的兄弟元素的子元素<a>移除"current"样式       
		.parent().siblings().children("a").removeClass("current")   
		//它们的下一个元素隐藏     
		.next().hide();                
		return false;
	});
})

jq对象和dom对象

jq对象是通过jq包装dom对象后产生的对象,在jq对象中无法使用dom对象的任何方法， dom对象的也不能使用jq里的方法。

$("#foo").html()// 获取id为foo的元素，这里的html()是jq里的方法

//等价于 js
document.getElementById("foo").innerHTML;

jq对象和dom对象互相转换,jq提供两种转换方法： [index] , get(index)

var $test=$("#test");
var test=$test[0]; //第一种方法
var test=$test.get(0);  //第二种方法

dom对象转换jq对象 ，只需要用 $() 把dom对象包装起来

var test=document.getElementById("test")
var $test=$(test);

# 选择器

jq选择器有完善的处理机制

$("#test").css("color","red");
// 这里无需判断$("#test")，就算是不存在也不会报错，要是js就会报错而不能进行别的操作

这里$("#test")获取的永远是对象，即使网页上没有此元素，因为要用jq检查某个元素在网页上是否存在：

// 不能这样检查 因为他永远是对象
if( ("#test") ){
//do something
}

// 但我们可以判断他的长度
if($("#test")[0]){
//do something
}

# 层次选择器

选择器 描述 示例

$("prev~siblings") 选取prev元素之后的所有siblings元素 $("#two~div")

可以使用 next()方法来代替$("prev+next")选择器

siblings()方法与前后位置无关，只要是同辈节点都能匹配

//选取#prev之后的所有同辈div元素
$("#prev~div").css("backgruond","red");

//选取#prev所有的同辈div元素，无论前后位置
$("#prev").siblings("div").css("backgruond","red");

# 过滤器选择器

过滤规则与css中的伪类选择器语法相同都是冒号 ：开头。

过滤选择器可分为 基本过滤、内容过滤、可见性过滤、属性过滤、子元素过滤、表单对象属性过滤。

选择器 描述 示例
:first 选取第一个元素 $("div:first")
:last 选取最后一个元素 $("div:last")
:not(selector) 删除所有与选择器匹配的元素 $("input:not(myClass)")
:even 选取索引是偶数的所有元素，索引从0开始 $("input:even")
: odd 选取索引是奇数的所有元素，索引从0开始 $("input: odd")
:eq(index) 选取等于index的元素，索引从0开始 $("input:eq(1)")
:gt(index) 选取大于index的元素， $("div:gt(1)")
:lt(index) 选取小于index的元素， $("div:lt(1)")
:header 选取所有标题元素如h1 、h2.. $(":header")
:animated 选取当前正在执行动画的元素 $("div:animated")
:focus 选取当前获取焦点的元素 $(":focus")

2.内容过滤器

选择器 描述 示例
:contains(text) 选取含有文本内容为“text”的元素 $("div:contains("你好")")
:empty 选取不包含子元素或者文本的空元素 $("div:empty")
:has(sele) 选取含有选择器所匹配的元素的元素 $("div:has(p)")
:parent 选取含有子元素或者文本的元素 $("div:parent")

3.可见性过滤器

选择器 描述 示例

:visible 选取所有可见的元素 $("div:visible")

4.属性过滤选择器

选择器 描述 示例
[attribute] 选取拥有此属性的元素 $("div[ id ]")
[attribute=value] 选取属性等于value的元素 $("div[title=test ]")
[attribute!=value] 选取属性不等于value的元素 $("div[title!=test ]")
[attribute^=value] 选取属性以value开始的元素的元素 $("div[title^=test ]")
[attribute$=valee] 选取属性以value结束的元素的元素 $("div[title$=test ]")
[attribute*=value] 选取所有属性含有value的的元素 $("div[title*=test ]")
[attribute~=value] 选取属性用空格分隔的值中包含一个给定值的元素 $('div[title~="uk" ]')
[attribute1] [attribute2] [attributeN] 用属性选择器合并成一个复合属性选择器 $("div[id][title$='test']")

5.子元素过滤选择器

选择器 描述 示例
:nth-childe(index/even/odd/equation) 选取每个父元素下的第index个子元素或者奇偶元素 :eq(index)是从0算起的只匹配一个元素 :nth-child(index)是从1算起的匹配所有符合条件的元素
:first-child 选取每个父元素的第一个子元素 $("ul li:first-child") 选取每个ul中第一个li元素
:last-child 选取每个父元素的一个子元素 $("ul li:last-child") 选取每个ul中最后一个li元素
: only-child 如果某个元素是它父元素中唯一的子元素，那么将会被匹配 $("ul li: only-child") 在ul中选取是唯一子元素的 li
:nth-child(even) //选取么个父元素下的索引值是偶数的元素
:nth-child(odd) //选取么个父元素下的索引值是奇数的元素
:nth-child(2) //选取么个父元素下的索引值是2的元素
:nth-child(3n) //选取么个父元素下的索引值是3的倍数的元素（n从1开始）
:nth-child(3n+1) //选取么个父元素下的索引值是(3n+1)的元素（n从1开始）

# 自定义动画

只有三种动画是不能满足需求的，可以用animate()方法来自己定义

通常配合position:absolute;来使用抽象理解，width、height(放大缩小)，left、top(用来位移的)

animate(params,speed,callback)：

  1. 一个包含样式属性及值的映射
  2. 速度参数， 可选
  3. 动画完成是执行的函数， 可选

    })

$(function(){
    $("#test").click(function(){

$(function(){
    $("#test").click(function(){

        })
    })
})

停止动画

停止元素的动画

stop([clearQueue].[gotoEnd]) :这来两个参数都是可选的：

clearQueue表示是否清空未执行完的动画队列

gotoEnd表示是否直接将正在执行的动画跳转到末状态

直接使用stop()方法会立即停止 当前正在进行的动画，如果接下去还有动画等待继续进行，就会在当前状态下 开始接下来的动画。

举例： 在一个元素绑定一个hover事件，用户快速移动鼠标不断的切换动画，这样就会在动画还没执行完成就移出了光标，那么移出光标事件就会 被放进动画队列中。 我们可以用stop()方法在停止当前动画（移入时的动画），这样他就会自动执行下面的动画（移出时的动画）

$("#panel").hover(function(){
    $(this).stop(true) //清空未执行完的动画队列
        .animate({heiht:"150px",width:"300px" },200);
},function(){
     $(this).stop(true,true) // 直接跳到末尾
        .animate({heiht:"22px",width:"60px" },200);
})

    //如果没有，就加入动画
}

# ajax

  1. 通过XMLHttpRequest对象向服务器端请求提交/接收数据
  2. 结合脚本不用刷新页面就能更页面
  3. 按需请求数据，能减轻服务器和宽带的负担

XMLHttpRequest对象 ajax的核心是XMLHttpRequest对象。它是ajax实现的关键===》发送异步请求、接收响应、执行回调都是通过它来完成的。

# jq封装的ajax()

第一层是$.ajax()

第二层是load()、$.get()和$.post()

# 第一层

$.ajax(Object)

Object参数：必填的四个参数 url、 type 、 datatype 、success

$(document).ready(function () {
    jQuery("#clearCac").click(function () {
        $.ajax({
            url: "/Handle/Do.aspx",
            type: "post",
            data: {id: '0'},
            dataType: "json",
            success: function (msg) {
                alert(msg);
            },
            error: function (XMLHttpRequest, textStatus, errorThrown) {
                alert(XMLHttpRequest.status);
                alert(XMLHttpRequest.readyState);
                alert(textStatus);
            },
            complete: function (XMLHttpRequest, textStatus) {
              //不管成功失败都会回调
            }
        });
    });
});
