# StringBuilder

提供一个与 StringBuffer 兼容的 API方法等都一样，但不保证同步， 线程不安全，但速度块，常用来字符串拼接。

public static void main(String[] args) {
    int[] arr = {1, 2, 3, 4};
    toSting(arr);//toSting的实现思路
}

public  void toSting(int[] arr) {
    StringBuffer buffer = new StringBuilder();
    buffer.append("[");
    for (int i = 0; i < arr.length; i++) {
        if (i == arr.length - 1) {
            buffer.append(arr[i]).append("]");
        } else {
            buffer.append(arr[i]).append(",");
        }
    }
    System.out.println(buffer);//[1,2,3,4]
}

# entrySet()

  • 键值对 映射关系，调用map集合方法entrySet()将集合中的映射关系对象存储到Set集合
  • Map.Entry<K,V> 映射项（键-值对）。
Map<Integer, String> map = new HashMap<Integer, String>();
map.put(1, "a");
map.put(2, "b");

//创建内部类对象=》外部类.内部类=new
Set<Map.Entry<Integer, String>> set = map.entrySet();

for (Map.Entry<Integer, String> i : set) {
    System.out.println(i);//1=a 2=b
    System.out.println(i.getKey());//1 2
    System.out.println(i.getValue()); //a b
}

用例

     c.set(2888, 5, 5); //直接设置2888年5月5日
}

}
