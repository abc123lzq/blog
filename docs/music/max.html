# 概念

# 波形和相位

  1. 声音都是有波形的。

  2. 当多个波形进行合并的时候就有相位概念,他会相互叠加也会相互抵消。

# 采样率和比特率

参数越高质量越好

  1. 采样频率是指采样速度
11025Hz //电话音质采样率
32000Hz //数码视频采样率
44100Hz //音频CD采样率
  1. 比特率是指每秒传送的比特数
96kb/sFM //广播质量 
128kb/s //普通MP3质量
1411kb/s16位 //CD质量

# 电平和声像

  1. 电平是指音量，以db为单位。

  2. 声像是指声音的发声位置，上下左右等等。

# 过载&削波

  1. 过载是指声音的电平信号，超出了阈值的最大电平。在电平表中0相当于最大阈值，过载就会失真
  1. 削波是因为过载的部分被消掉，所以产生了音频爆掉的听感。 一般采用限制器(Waves L2)来防止过载。

# 插件效果器

业界比较出名的插件套装

  1. Waves 推荐
  2. Fabfilter
  3. 插件联盟Plugin Alliance

# 均衡器

混音中最重要的效果器之一，能调整各频段信号的增益量,和衰减多余频率，同时还能塑造音色

# 压缩器

对于忽大忽小的声音,采用压缩来控制电平同时也会影响音色

主要参数

  1. 阈值:触发的值
  2. 压缩比
  3. 启动时间:压缩到预定电平所需要的时间
  4. 释放时间：恢复到不工作所需要的时间

# 限制器

和压缩器有点像，限制器是为了将信号峰值降低到阈值以下，也常用于提高我们整体混音的响度

主要参数

  1. 阈值：触发的值
  2. 输出上限:能被输出的最高电平
  3. 释放时间：恢复到不工作所需要的时间
  4. 抖动：给声音添加一层白底噪

# 门限器

常用于消除噪音

# 混响器

构造空间感，美化音色。

  1. 房间混响：时间段，密度大。
  2. 大厅混响： 密度偏低，空间大，时间长。
  3. 板式混响：空间距离感不明确，高频较为华丽。

# 延迟器

让声音不断复制达到延迟效果

主要参数

  1. 延迟时间
  2. 会受 Feedback：延迟会被重复的数值

常用ping-ping延迟，如：8/1ping-pong & 4/1延迟，在配点混响。

# 高音修正器

用来修复音准的

  1. Waves Tune 优点：兼容性好操作方便 缺点： 界面小,音质差

  2. Auto Tune 优点：音质好，容易上手 缺点：流畅版本不支持主流宿主，最新版本掉帧闪退

  3. Meloadyne 优点：功能多，可修音、修节奏，音质还可以 缺点：操作麻烦
