# mongodb安装

# 安装流程图

# 启动MongoDB服务

net start MongoDB

# 关闭MongoDB服务

net stop MongoDB

# 配置进程守护

  1. 在mongodb安装的目录下面，找到data文件夹，然后在里面创建db文件夹和log文件夹
  2. 在mongodb安装目录创建mongo.config打开输入：
<!--安装的目录如：E:\mongodb-->
dbpath=<安装的目录>\data\db
logpath=<安装的目录>\data\log\mongodb.log
  1. cmd中跳转到mongodb的bin目录。然后输入
mongod –config E:\mongodb\mongo.config --install –serviceName “MongoDB”
