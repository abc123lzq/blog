# 服务类目不符合

  1. 服务类目"房地产-装修/建材_"与你提交代码审核时设置的功能页面内容不一致: (1):小程序页面内容涉及账户充值服务，需补充商家自营-预付卡销售-发行方类目。请在基础信息处申请该类目，通过资质审核并在配置功能页添加符合该类目的功能页面。 请根据上述原因对小程序进行修改，并重新提交代码审核。

  2. 你好，你的小程序涉及商家入驻/店铺街入口，请补充选择：电商平台类目，或者建议去掉该模块内容，改成商家自营模式。 (1).由于电商类目需要电商资质，第三方入驻审核也比较严格，我们可以绕过，因为个人自营的审核比较不严格。 (2).更改类目的时候经常会在提交版本审核的时候提示，更改较大包含类目等需要7-12天的审核。

  3. 在线观看 需要添加文娱-视频类目

  4. 自行生成的内容的发布/分享/交流服务 添加 社交-社区/论坛类目

# 体验功能不符合

  1. 你好，小程序帐号登录功能暂未符合规范要求，请在用户了解体验小程序功能后，再要求用户进行帐号登录。请整改后再重新提交审核，具体登录规范及整改可参

  2. 小程序测试账号不符合规则: (1):你好，小程序功能无法完整体验，请在下一次提交审核时提供有效测试信息或将小程序使用流程、运行录屏上传至云盘，在小程序测试信息填写栏/版本描述/备注处附上测试信息或云盘账号密码协助完成审核。

  3. 小程序可用性和完整性不符合规则: (1):你好，你的小程序点击【班级管理】【活动管理】服务核心功能无法完整体验，请完善服务功能后再提交代码审核。（测试环境wifi, IOS 12.2，微信7.0.4，IPHONE 6）
