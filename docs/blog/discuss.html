# 评论插件

# utterances

是基于github issue，拥有多种主题的开源免费的插件

# 安装

打开链接直接点击安装

然后推荐选对应的github仓库进行授权就可以了

# 仓库配置

访问配置页面就可进行配置

# utteranc配置

引入脚本就可以在页面使用评论功能了

<script src="https://utteranc.es/client.js"
        repo="[ENTER REPO HERE]" 
        issue-term="title" 
        theme="github-light"
        crossorigin="anonymous"
        async>
</script>

md文档无法直接引入js，我可以用vue组件引入就行了,到此完毕。

<template>
    <div id="comment"></div>
</template>
<script>
    export default {
        name: 'Utterances',
        mounted: function () {
            this.initValine()
        },
        methods: {
            initValine() {
                // 建立脚本以及属性
                const utterances = document.createElement('script');
                utterances.type = 'text/javascript';
                utterances.async = true;
                // utterances.setAttribute('issue-number', this.id)
                utterances.setAttribute('issue-term', 'title')//标题：取当前页的
                utterances.setAttribute('theme', 'github-light')
                utterances.setAttribute('repo', `abc123lzq/itdoc`) //这个厂库地址：
                utterances.crossorigin = 'anonymous';
                utterances.src = 'https://utteranc.es/client.js';

                // comment 是要插入评论的地方
                window.document.getElementById('comment').appendChild(utterances);
            }
        }

    }
</script>
