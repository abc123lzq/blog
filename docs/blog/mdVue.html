# md里面引入组件
🔥hot

*所有在 .vuepress/components 中找到的 .vue 文件将会自动地被注册为全局的异步组件，如：

.
└─ .vuepress
   └─ components
      ├─ demo.vue
      ├─ OtherComponent.vue
      └─ Foo
         └─ Bar.vue

你可以直接使用这些组件在任意的 Markdown 文件中（组件名是通过文件名取到的）：

注意组件名有区分大小写，新增的组件需要重启一下服务器,初始化后组件里的内容就可以热更新了

<demo></demo>
<OtherComponent></OtherComponent>
<Foo-bar></Foo-bar>

# 自定配置使用预处理器

已经内置相关的 webpack 所以我们可以配置：sass、scss

npm add -D sass-loader node-sass

然后你就可以在 Markdown 或是组件中使用如下代码：

<style lang="sass">
  .title
    font-size: 20px
</style>
