# vConsole

vConsole腾讯开发的轻量级移动端调试工具

# 安装

npm install vconsole

# 使用

<script>
import 'path/to/vconsole.min';
  // 初始化
  var vConsole = new VConsole();
  console.log('Hello world');
</script>

游览项目就能看到 vConsole 图标点击后就是控制台

# 小程序自带

小程序开发和体验环境自带vConsole功能。

例如：当真机调试时候手机扫码进入调试的时候编辑器也会打开自带的调试工具进行实时调试, 点击手机页面的是...呼出功能里也有调试功能 内嵌的就是vConsole


TIP

vConsole轻量级挺好的用

有的时候开发环境没问题，但扔到线上出来问题，还是得用 fiddler 或whistle 他拥有更多功能不止于打印
