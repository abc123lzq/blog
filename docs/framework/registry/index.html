# verdaccio

由于sinopia作者以停止维护了，所以verdaccio团队Fork分支出来进行维护。

安装CLI

npm install -g verdaccio

启动

verdaccio

设置npm从verdaccio源下载

npm set registry http://localhost:4873/

pm2守护verdaccio进程

pm2 start verdaccio

# 发布npm包

创建新用户

1.确保自己已经切换到verdaccio仓库

➜  ~ nrm ls

  npm ---- https://registry.npmjs.org/
  cnpm --- http://r.cnpmjs.org/
  taobao - http://registry.npm.taobao.org/
  nj ----- https://registry.nodejitsu.com/
  rednpm - http://registry.mirror.cqupt.edu.cn
  npmMirror  https://skimdb.npmjs.com/registry
* verdaccio  http://192.168.1.200:4873/

2.运行npm adduser,填写信息，注册账号。如果已经有账号，直接运行npm login即可。

npm adduser
Username: test
Password:test

发布之前要用下面命令登录：

npm login

进入我们要发布的包根目录，然后执行初始化命令创建 package.json （已经初始化的话，跳过这步）：

npm init

然后在根目录执行下面命令进行发布：

npm publish

发布成功后，你就可以安装自己发布的包了

# 配置文件

在运行verdaccio时候有打印出地址，一般在：C:\Users\Administrator\AppData\Roaming\verdaccio

配置verdaccio从淘宝镜像下载，一般默认就是配置淘宝镜像

uplinks:
  npmjs:
    url: http://registry.npm.taobao.org/
